## Recommended Path

### Phase 1 — Web3 Gap-fill (Week 1–2)
You're already strong on AI. Fill the Web3 side before going into the Bridge.

| Topic | Handbook Section | Priority |
|-------|-----------------|----------|
| Wallet as identity & signing entry | Wallet | High |
| Smart contract basics | Smart Contract | High |
| Account Abstraction & Agent permissions | Account Abstraction | High |
| Contract/permission security risks | Security | High |
| Oracle as a risk surface | Oracle | Medium |
| DeFi: assets, liquidity, risk propagation | DeFi | Medium |

> Skip: Network, Cryptography (read only if concepts feel unclear)

### Phase 2 — AI × Web3 Bridge (Week 3–4)
This is your core target area.

| Topic | Handbook Section | Why It Matters for You |
|-------|-----------------|----------------------|
| How chain state enters Agent context | Chain-aware Context | Foundation for all demos |
| Calling RPC, wallet, contract tools | Web3 Tool Use | Build your first demos here |
