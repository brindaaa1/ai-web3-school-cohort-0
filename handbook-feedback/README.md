# Handbook Feedback

This folder collects issues, suggestions, and corrections found while studying the [AI × Web3 School Handbook](https://aiweb3.school/zh/handbook/).

## How to file feedback

Create a new file named `YYYY-MM-DD-topic.md` using the template below.

---

## Feedback Template

```
# Feedback — TOPIC

- Date: YYYY-MM-DD
- Handbook page: [link]
- Type: typo / unclear concept / outdated resource / structure suggestion / missing content

## Issue

> Describe the problem clearly.

## Suggested Fix

> What should it say or how should it be restructured?

## Source

> Where did you encounter this? (reading, exercise, demo, discussion)
```
